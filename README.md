# Cross-Spectral-Tree-Canopy-Damage-Paper

### Data Dictionary

| Variable (Original Spanish) | Type | English Translation / Description |
| :--- | :--- | :--- |
| **DAP** | Continuous (float) | Diameter at Breast Height (cm), measured at 1.30 m. |
| **Altura** | Continuous (float) | Total tree height (m), estimated using a hypsometer. |
| **Especie** | Categorical | Tree species (taxonomic or common name). |
| **Tronco y ramas** | Categorical | Structural condition assessment of the trunk and branches. |
| **Fenologia** | Categorical | Phenological status (e.g., vegetative, flowering). |
| **Plagas** | Categorical | Presence/absence of pests or diseases. |
| **Cochinillas** | Categorical | Presence/absence of scale insects (*Coccoidea*). |
| **Follaje** | Categorical | **Target label**: Canopy status (Mild damage, Moderate damage, Total defoliation, Tree death). |
